# 🛡️ Multi-Cipher Encryption Tool (C# GUI)

A professional desktop application built with **C# and .NET** that provides a user-friendly interface for encrypting and decrypting text using various classical and modern cryptographic algorithms.

## 🚀 Features
The application supports **7 different encryption/decryption algorithms**, allowing users to explore how different ciphers work:

* **Caesar Cipher:** Simple substitution by shifting characters.
* **Monoalphabetic Cipher:** Substitution using a fixed alphabet mapping.
* **Vigenere Cipher:** Polyalphabetic substitution using a keyword.
* **Playfair Cipher:** Diagram-based encryption using a 5x5 matrix.
* **Rail Fence Cipher:** A type of transposition cipher that follows a "zigzag" pattern.
* **Row Transposition:** Rearranges characters based on a numerical key.
* **Feistel Cipher:** A symmetric structure used in many block ciphers (implementing the core logic).

## 🛠️ Technologies Used
* **Programming Language:** C#
* **Framework:** .NET Framework (WinForms/WPF)
* **Concepts:** Information Security, Cryptography, GUI Design, Algorithm Implementation.

## 💻 How to Use
1.  **Select Algorithm:** Choose one of the 7 ciphers from the menu.
2.  **Input Text:** Enter the plain text you want to encrypt or the cipher text to decrypt.
3.  **Enter Key:** Provide the necessary key (numeric or string) required for the specific algorithm.
4.  **Process:** Click **Encrypt** or **Decrypt** to see the results instantly.
